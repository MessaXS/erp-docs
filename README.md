# ERP — Документация для сотрудников

Здесь собраны пошаговые инструкции для каждой роли в ERP-системе.
Каждая инструкция описывает один конкретный процесс: что нажать, что заполнить, что получится.

## Как пользоваться

1. Найдите свою роль в таблице ниже.
2. Откройте README роли — там карта всех процессов.
3. Пройдите по процессам в указанном порядке.

## Формат документации

- Один файл = один процесс (одна задача)
- Каждый шаг = одно действие
- После каждого шага — место для скриншота
- Скриншоты хранятся в папке `img/` рядом с процессом
