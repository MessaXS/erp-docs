# Менеджер отдела реализации (ManagerSales)

## Кто это

Менеджер отдела реализации — это человек, который работает с клиентами.

Он:
- Добавляет новых клиентов в систему
- Создаёт заказы и заявки на работников
- Следит, чтобы все смены были закрыты
- Ведёт учёт расходов и начислений

Другое название этой роли: **менеджер по работе с заказчиками**, **менеджер ОРЗ**.

## Важно помнить

- **Заверение = блокировка.** После заверения начисления его нельзя изменить (кроме комментария).
- **Закрытый период.** Если дата операции попадает в закрытый период — менять ничего нельзя.
- **Серые кнопки.** Если кнопка серая и не нажимается — значит, у вас нет прав на это действие. Функция существует, но доступна другой роли.
